About me.

I am currently a 16 year old high school student from Kiel, Germany.

I was in 6th grade when I first got in touch with programming and coding. When the COVID-19 pandemic started, I decided to learn the basics of JavaScript. However, I quickly realized I wanted to learn more about mathematical calculations and creating projects for that I could focus more on the logical side of things, rather than a user interface or user experience.

I therefore learned Python and started creating simple programs, for instance a binary-decimal converter. After that, I built a TicTacToe artificial intelligence to play against, feel free to check out the code on GitHub.

After a break I now enrolled in a course about Data Analytics offered by Google and Coursera in order to learn more about data in general, big data, and data visualization.
